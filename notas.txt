EXPLICAÇÃO CLIENT SIDE / SERVER SIDE

REQUEST COMPLETA

DIFERENÇA CLIENT SIDE / SERVER SIDE

CLIENT SIDE:
HTML, CSS3 E JS BASICO

RESPONSE: SERVIDOR --> INTERNET --> CLIENTE (COPIA DOS ARQUIVOS)

SERVER SIDE
PODE TER CLIENT SIDE E SCRIPTS

RESPONSE: SERVIDOR(ARQUIVOS NÃO PODEM SER ENVIADDOS ENTÃO GERA RESULTADOS) --> INTERNET --> CLIENTE(COPIA DOS ARQUIVOS SOMENTE DEPOIS DO PROCESSO DO SERVIDOR)

FRONT-END / BACK-END / FULLSTACK

FRONT-END É ESPECIALIZADO EM COISAS QUE VÃO RODAR NO CLIENTE 
BACK-END É ESPECIALIZADO EM PROGRAMAÇÃO QUE VÃO RODAR NO SERVIDOR
FULLSTACK É ESPECIALIZADO EM AMBOS
